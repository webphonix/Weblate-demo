### WebLate:
**Best javascript library to load dynamic template.**
WebLate is similar to JQuery, react and angular. 

## Advantages:
* reusability of code
* time saving
* light weight code
* dynamic data entry
* less code more work
* easy to use

## Installation:
npm i weblate
